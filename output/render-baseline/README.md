# Render Baseline Checklist

Capture and store screenshots for these scenes during manual QA:

- empty discovered terrain
- dense terrain with roads, bridges, and building overlays
- many heroes/entities on screen
- overlays active: hover, task menu, path preview, reach outline
- particles and effects active
- multiple camera positions near screen edges

Record each screenshot with:

- camera position
- quality profile
- active overlays/effects
- notes on any intentional visual deviation
